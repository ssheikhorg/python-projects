## Installation Instructions

Below are some instructions that are worth making note of:

## Install

You must have [python](https://www.python.org/downloads/) installed, then:

```bash
python3 -m venv venv
```

Activate the virtual environment:

```bash
source venv/bin/activate
```

Now that you have python installed & venv activated, install python packages.

```bash
pip install -r requirements.txt
```

Then we can run the application:

```bash
python main.py
```
